Picking Domain Fix

Improves the domain when selecting a partner in pickings.

Translated to all Odoo languages

Description

Only main partners and shipping addresses shown

When you select a partner for an incoming/outgoing stock picking, you see companies as well as their contacts. With this module installed, you still will see partners with no parent but only the contacts which are shipping addresses, which makes more sense and makes the dropdown get rid of useless options. In internal pickings, only the current company and their shipping addresses will be available to select if the dropdown of partners was shown.

Installation

To install this module, you need to:
  • You must have installed or available to install the module stock.

Usage

To use this module, you need to:
  • Go to Inventory > Operations > All Transfers, and create / edit any picking.
  • Open the dropdown of the Partner field, and check that you only see partners with no parent and the contacts which are shipping addresses (in case of incoming/outgoing pickings).
